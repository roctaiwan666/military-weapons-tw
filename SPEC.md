# 國軍現役武器裝備資訊網 - 專案規格說明

## 專案目標與願景
建立一個現代化、資訊豐富的前端網站，展示中華民國國軍現役武器裝備的資訊。

### 核心目標
1. 提供清晰易用的三軍裝備資訊展示
2. 實現良好的使用者體驗和互動效果
3. 確保資訊的準確性和時效性
4. 支援多裝置瀏覽體驗

### 目標受眾
- 對國防軍事有興趣的民眾
- 軍事研究人員
- 新聞媒體工作者

## 技術規格

### 使用技術
- HTML5 - 網頁結構
- CSS3 - 頁面樣式與動畫
- JavaScript (原生) - 互動功能實作
- CDN 引入的外部套件：
  - Bootstrap 5.3.0 - UI 框架，用於實現響應式設計
  - AOS 2.3.1 (Animate On Scroll) - 實現滾動動畫效果
  - Swiper 10.0.0 - 實現圖片輪播展示功能

### 瀏覽器支援
- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)

### 檔案結構
```
military-weapons-tw/
├── index.html       # 主頁面，包含所有的頁面結構和內容
├── css/
│   └── style.css   # 自訂樣式，包含響應式設計和動畫效果
├── js/
│   └── main.js     # 自訂互動功能，包含滾動動畫和導航功能
└── images/         # 圖片資源目錄
    ├── hero-bg.jpg  # 首頁橫幅背景（待新增）
    ├── cm11.jpg     # 勇虎戰車圖片（待新增）
    ├── kdx.jpg      # 康定級巡防艦圖片（待新增）
    └── f16v.jpg     # F-16V戰機圖片（待新增）
```

### 主要功能模組
1. 導航系統
   - 響應式導航欄
   - 平滑滾動
   - 自動高亮當前區段

2. 裝備展示系統
   - 卡片式裝備資訊
   - 圖片輪播展示
   - 分類瀏覽功能

3. 搜尋和篩選系統（計畫中）
   - 關鍵字搜尋
   - 分類篩選
   - 標籤過濾
